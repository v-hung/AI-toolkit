# Guidelines

## Replies

- Be concise in your explanations - focus on what's important rather than explaining obvious details.

## Language Rules

Conversation Language: Vietnamese  
Documentation Language: English
Code Language: English

Rules:

- All generated documentation must be written in English
- Chat responses can be in Vietnamese
- Code comments must be in English

## Document Rules

- Use **Markdown formatting**
- Use concise and professional documentation style
- Do not invent features not mentioned in the brief
- Keep the overview under 150 words
