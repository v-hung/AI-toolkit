# Use Case

## Use Case ID

UC-001

## Use Case Name

User Login

## Actors

- User
- System

## Description

User đăng nhập vào hệ thống.

## Preconditions

- User đã có account.

## Postconditions

- User đăng nhập thành công.

## Main Flow

1. User nhập email
2. User nhập password
3. User click login
4. System xác thực
5. System redirect dashboard

## Alternative Flow

### Invalid Password

1. System hiển thị lỗi

## Exceptions

- System unavailable
